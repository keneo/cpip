# Cpip

## Requirements

- [Java SE](http://www.oracle.com/technetwork/java/javase/overview)
- [Gradle](http://www.gradle.org)

## Usage/demo

```
git clone https://github.com/keneo/cpip
cd cpip
gradle run
```
